## USAGE

```
$ git clone https://github.com/misebox/tetrominoes-with-pygame2
$ cd tetrominoes-with-pygame2
$ pip install -r requirements.txt
$ python tetro.py
```

## KEYBOARD MAPPING

|key|description|
|:-|:-|
|ESCAPE|QUIT GAME|
|h, ←|MOVE LEFT|
|l, →|MOVE RIGHT|
|j, ↓|MOVE DOWN|
|R, e, k|ROTATE RIGHT|
|r, E, K|ROTATE LEFT|
|SPACE|DROP|
